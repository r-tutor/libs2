Defunct Functions in the Package geoR

Description

The functions listed here are no longer part of the package geoR as they are not needed (any more).

Usage

geoRdefunct()

Details

The following functions are now defunct:

  1. olsfitfunctionality incorporated by variofit. From geoR_1.0-6.

  2. wlsfitfunctionality incorporated by variofit. From geoR_1.0-6.

  3. likfit.oldfunctionality incorporated by likfit. From geoR_1.0-6. The associated functions were also made defunct:
    likfit.nospatial, loglik.spatial, proflik.nug, proflik.phi, proflik.ftau.

  4. distdiagfunctionally is redundant with dist.
