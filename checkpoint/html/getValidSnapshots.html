getValidSnapshots {checkpoint}R Documentation

Read list of available snapshot dates from MRAN.

Description

Returns vector of available dates from MRAN or local MRAN repository.

Usage

getValidSnapshots(mranRootUrl = mranUrl())

Arguments

mranRootUrl

MRAN root. This can be a URL, e.g. https://mran.microsoft.com/snapshot/ or the path to a local MRAN repository, e.g.file:///local/path

Value

Character vector with dates of valid snapshots
