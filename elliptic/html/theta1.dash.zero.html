Derivative of theta1

Description

Calculates theta1' as a function of either m or k

Usage

theta1.dash.zero(m, ...)
theta1.dash.zero.q(q, ...)

Arguments

m

real parameter

q

Real parameter

...

Extra arguments passed to theta1() et seq, notably maxiter

Author(s)

Robin K. S. Hankin
