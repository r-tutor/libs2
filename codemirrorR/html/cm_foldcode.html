cm_foldcode {codemirrorR}R Documentation

Code Folding for CodeMirror

Description

Add code folding for your codemirror htmlwidget. This is designed for easy piping with magrittr or pipeR.

Usage

cm_foldcode(cm_widget = NULL, fold_mode = NULL)

Arguments

cm_widget

codemirror htmlwidget

fold_mode

character for the type of folding. Valid options are "brace-fold", "comment-fold", "indent-fold", "markdown-fold", and "xml-fold".

Value

codemirror htmlwidget
